== Description ==

This plugin gives your biggest fans another way to keep up with your digital download downloads with free and premium download type. 
With PayPal integration supports makes your downloaded file easy to set to premium type. The download counter will helps you monitor 
your downloaded files, storing the user IP adress or user ID for premium download.

<h4>Key Features & Options</h4>
<ul>
<li>No additional post types added.</li>
<li>Support for all custom post types.</li>
<li>PayPal and multi currency supports.</li>
<li>Shortcodes supports.</li>
<li>Premium and free download type.</li>
<li>List table for purchased download.</li>
<li>List table for all downloads.</li>
<li>Registered users only or public download type.</li>
<li>Fancy URL for downloading item.</li>
<li>Download counter</li>
<li>More plugin settings</li>
</ul>

<h4>Using Plugin</h4>
<ul>
<li>Go to your post, page or post type edit screen and set Tomatoes post meta option.</li>
<li>Give a title, description, price (for premium) and choose the file for download in the metabox.</li>
<li>Paste <code>[download]</code> anywhere in your post content.</li>
<li>That's it, simply is the best.</li>
</ul>
